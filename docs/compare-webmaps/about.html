Comparing webmaps, locales, themes and API versions in 4x JSAPI

This app shows two map views of a webmap displayed with ArcGIS API for JavaScript 4.x. You can specify different settings for each of the view. Each parameter either takes a single string (for same value to both maps), or a comma separated string with two values.

  1. different API versions
    • Example: https://bsvensson.github.io/various-tests/compare-webmaps/index.html?version=4.18,4.21
    • Default is "4.20" on left and "next" on right.
  2. different API builds
    • Example: https://bsvensson.github.io/various-tests/compare-webmaps/index.html?api=https://jsdev.arcgis.com/next/,https://jsqa.arcgis.com/next/
  3. different webmaps (including from different environments)
    • one webmap: https://bsvensson.github.io/various-tests/compare-webmaps/index.html?webmap=84bf6b30a1d740fba9f19774e00f0466
    • two webmaps: https://bsvensson.github.io/various-tests/compare-webmaps/index.html?webmap=84bf6b30a1d740fba9f19774e00f0466,b6943559f896451698274036447f656f
    • devext webmap: https://bsvensson.github.io/various-tests/compare-webmaps/?webmap=c03c8685721b4b7a99f0968765277530&portal=https://devext.arcgis.com/
    • one devext webmap and one production webmap: https://bsvensson.github.io/various-tests/compare-webmaps/?webmap=c03c8685721b4b7a99f0968765277530,84bf6b30a1d740fba9f19774e00f0466&portal=https://devext.arcgis.com/,https://www.arcgis.com
  4. different locales
    • Example: https://bsvensson.github.io/various-tests/compare-webmaps/index.html?locale=sv,ja
    • See list of locates at https://developers.arcgis.com/javascript/latest/localization/">https://developers.arcgis.com/javascript/latest/localization/
  5. different themes
    • Example: https://bsvensson.github.io/various-tests/compare-webmaps/index.html?theme=light,dark

Tips & tricks

  • Replace index.html with webmap.html in the URL and you'll get the "one map" experience (instead of side-by-side).

Known differences

  • The colorful themes are deprecated as of 4.19 (see console message).
